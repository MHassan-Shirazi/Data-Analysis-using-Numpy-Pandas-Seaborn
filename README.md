# Data Analysis using NumPy, Pandas, and Seaborn

[![GitHub stars](https://img.shields.io/github/stars/MHassan-Shirazi/Data-Analysis-using-Numpy-Pandas-Seaborn?style=social)](https://github.com/MHassan-Shirazi/Data-Analysis-using-Numpy-Pandas-Seaborn/stargazers)

## Overview

This repository contains multiple data analysis projects conducted using Python libraries **NumPy**, **Pandas**, and **Seaborn**. Each project explores different datasets, performs data cleaning, visualization, and extracts actionable insights.

## Projects Included

| Dataset | Description |
|---------|-------------|
| **Bank Dataset** | Analysis of customer banking data to identify patterns and trends. |
| **Financial Dataset** | Insights into financial transactions and trends for decision-making. |
| **Sales Dataset** | Exploration of sales data to understand revenue and product performance. |
| **SuperMart Dataset** | Retail dataset analysis to identify sales patterns and customer behavior. |
| **Titanic Dataset** | Exploratory data analysis on the Titanic dataset, including survival analysis. |

## Features

- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Visualization using Seaborn and Matplotlib  
- Insight extraction and reporting  
- Support for multiple types of datasets (banking, sales, finance, retail, historical datasets)  

## Technologies Used

- **Python**  
- **NumPy** – For numerical computations  
- **Pandas** – For data manipulation and analysis  
- **Seaborn** – For data visualization  
- **Matplotlib** – For plotting graphs
